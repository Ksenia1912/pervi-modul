Данная программа реализует метод градиентного спуска для минимизации функций двух переменных (x1 и x2). Программа позволяет вводить математические функции, начальные значения и параметры алгоритма, а затем отображает результаты вычислений.

------- Данные должны вводиться в таком формате: -------
-----------------------------------------------------------------------------

1. Ввод функции. Вводится математическое выражение с переменными x1 и x2. 

Допустимые элементы:
- Переменные: x1, x2
- Числа: целые и десятичные
- Математические операторы: "+", "-", "*", "/"
- Математические функции:
- Скобки "( )" для группировки выражений
- sin(x) - синус угла в радианах
- cos(x) - косинус угла в радианах
- tan(x) - тангенс угла в радианах
- exp(x) - e в степени x
- log(x) - натуральный логарифм
- sqrt(x) - квадратный корень
- abs(x) - абсолютное значение

Примеры корректных функций:
x1*2 + x2*2
sin(x1) + cos(x2)
(x1 - 1)*2 + (x2 - 2)*2

2. Ввод начальных значений 
- Два числовых значения через пробел 
- Десятичные числа указываются через запятую
- Отрицательные числа указываются с минусом

Примеры:
0 0
1,5 -2,3
-3 4,7
2.0 -1,5

3. Ввод точности . 
Точность определяет критерий остановки алгоритма. Алгоритм завершается, когда разница значений функции между итерациями становится меньше этого значения.
Рекомендуемые значения: 0.001, 0.0001, 0.00001

Примеры:
0,001
0,0001
1e-5

4. Ввод шага обучения 
Рекомендации:
- Слишком большой шаг может привести к расхождению
- Слишком маленький шаг замедляет сходимость
- Обычно в диапазоне 0.001 - 0.1

Примеры:
0,01
0,05
0,1

------- Особенности работы программы -------

Алгоритм градиентного спуска
1. Вычисляет значение функции в начальной точке
2. Рассчитывает градиент функции численным методом
3. Делает шаг в направлении антиградиента
4. Повторяет до достижения заданной точности

Ограничения
- Максимальное количество итераций: 10000
- Функция должна быть определена во всех точках траектории
- Программа ищет локальный минимум (ближайший к начальной точке)

Область результатов отображает:
- Исходную функцию
- Начальные значения
- Параметры алгоритма
- Найденные значения переменных
- Значение функции в найденной точке
- Количество итераций
- Время вычисления

------- Советы по использованию -------
Для лучшей сходимости:
1. Начинайте с простых функций для тестирования
2. Подбирайте шаг обучения экспериментально
3. Используйте различные начальные точки для поиска глобального минимума
4. Для "овражных" функций может потребоваться уменьшение шага
